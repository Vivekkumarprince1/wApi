# Get Solution ID from Meta

This guide walks you through obtaining a Solution ID from Meta to work as a Tech Provider with Gupshup.

## 1. Meta Tech Provider Signup
- **Business Portfolio**: Use or create a Meta Business Account. Business verification is mandatory.
- **Create Meta App**: Create a 'Business' app on Meta and connect it to your Business Portfolio.
- **Create Partner Solution**:
  - In App Dashboard, go to **Use cases > WhatsApp > Customize**.
  - Agree to Tech Provider terms.
  - Select **Partner Solutions**.
  - Solution Name: Use `TechProviderName Gupshup OD` (no special characters).
  - Partner App ID: `340384197887925` (OneDirect/Gupshup).
  - Permissions: Select **ONLY MY PARTNER** for 'Send messages' (crucial for Gupshup functionality).

## 2. Verification & App Review
- **Business Verification**: Mandatory before app review.
- **App Review**: Required for `whatsapp_business_messaging` and `whatsapp_business_management` permissions.
- **Access Verification**: Final step to ensure no enforcement breaks.

## 3. Gupshup Mapping
- Once approved by Meta, register the Solution ID in the **Gupshup Partner Portal > Settings > Add new solution**.
- Default Solution IDs are applied during app creation and Go-Live.
