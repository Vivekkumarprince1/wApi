# Onboarding APIs

Enables Tech Providers to automate WABA onboarding for customers without using the Gupshup UI.

## Key APIs
- **Create App**: Creates a pre-linked Gupshup app.
- **Set Callback**: Configures the event webhook.
- **Generate Embed Signed Link**: Creates the link for the Meta Embedded Signup flow.
- **Update App**: Modifies template messaging and storage region settings.
- **Mark App for Migration**: Used when migrating from another BSP to sync templates.
- **Resend Verification Link**: Triggers verification emails to the end client.

## Prerequisites
- Approved Solution ID in Gupshup Partner Portal.
- Wallet configured in a self-serve or partner account.

[Source: Gupshup Documentation](https://partner-docs.gupshup.io/docs/onboarding-apis)
