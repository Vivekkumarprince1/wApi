# API Reference: Get WABA INFO

Retrieves detailed information about a specific WhatsApp Business Account (WABA).

## Key Information Provided
- **Account Status**: (e.g., ACTIVE, DISABLED, RESTRICTED).
- **Messaging Capabilities**: Current limits and throughput capacities.
- **Phone Quality**: Quality rating (HIGH, MEDIUM, LOW, FLAG) and tier information.
- **Namespace**: The Meta-assigned namespace for the account's templates.
- **Ownership**: Verification of the BMID linked to the WABA.

[Source: Gupshup Documentation](https://partner-docs.gupshup.io/reference/get-waba-info)
