# ConnectSphere Career Portal

Next.js career portal for public jobs, candidate applications, and role-based internal hiring workflows.

## Local Development

```bash
npm i
npm run dev
```

The dev server uses port `3200` by default. If the app is already running, the dev script prints the active URL instead of failing with `EADDRINUSE`.

## Dev Credentials

Seeded role credentials are documented in [DEV_CREDENTIALS.md](./DEV_CREDENTIALS.md).

Keep these credentials for development only; do not expose them in the public UI.
