# Browser and device support

Target WCAG 2.2 AA and the latest two stable releases of Chrome, Edge, Firefox and Safari, plus current iOS Safari and Android Chrome. Support responsive layouts from 320 CSS pixels upward. Internet Explorer and embedded legacy webviews are unsupported.

Release checks cover keyboard-only navigation, visible focus, 200% zoom, reduced motion, screen-reader landmarks/forms, mobile management lists, upload recovery and critical authenticated journeys.
