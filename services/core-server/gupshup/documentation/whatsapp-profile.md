# WhatsApp Profile Display Name

This API allows partners to manage the display name for a WhatsApp application.

## Key Actions
- **Get Display Name**: Retrieves the current display name for an application.
- **Update Display Name**: Submits a new display name for review.

## Guidelines
- Display names must comply with **WhatsApp Commerce and Business Policies**.
- Changes typically trigger a review process by Meta.
- The status of the display name change can be tracked via the `pndn-event` in System Events.
