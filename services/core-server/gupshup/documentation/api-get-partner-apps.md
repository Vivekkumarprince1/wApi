# API Reference: Get Partner Apps

Retrieves a comprehensive list of applications linked to the partner account.

## Response Data
- **DockerDetails**: Includes current health status of the application's WhatsApp API client.
- **CappingDetails**: Usage counts and messaging limits (capping) for the application.
- **App Details**: Name, ID, category, and creation timestamps.

[Source: Gupshup Documentation](https://partner-docs.gupshup.io/reference/get_partner-account-api-partnerapps)
