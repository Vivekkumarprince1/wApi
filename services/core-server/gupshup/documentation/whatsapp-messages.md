# WhatsApp Messages & Conversations

## Conversations
Threads are 24-hour windows opened by either a free-form message or a template message. Conversations are the fundamental unit of WhatsApp pricing.

## Message Status Definitions
- **sent**: Message is in transit within Gupshup systems.
- **delivered**: Message reached the customer device.
- **read**: Customer viewed the message (if read receipts are enabled).
- **failed**: Delivery failed; details included in the callback.
- **deleted**: Customer deleted the message; system should mirror this deletion if required.
- **warning**: Issue with catalog items or other non-critical errors.

## Scaling
To increase 24-hour conversation limits, businesses must maintain high quality ratings and follow Meta's messaging limit tiers.
