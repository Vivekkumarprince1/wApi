# Generate Secret and Token

## API Client Secret
Used to generate a partner token via the Login API. 

### Generation Steps
1. Navigate to Settings page in Partner Portal.
2. Click **Create client secret**.
3. Choose expiry (can be set to "Never").
4. Click **Generate secret** and copy it immediately.

### Important Notes
- **Legacy Users**: Partners onboarded before 20th Nov 2024 use their portal password as the client secret by default. However, changing the password will require generating a new client secret via the settings page.
- **MFA**: MFA can be enabled for the partner account for added security.

[Source: Gupshup Documentation](https://partner-docs.gupshup.io/docs/generate-secret-and-token)
